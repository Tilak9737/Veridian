# Veridian

Veridian - Offline Bill Tracking | Electron, SQLite, Chart.js, HTML/CSS/JS 
**Tech Stack:** We built this platform using Electron for a powerful, secure desktop application and SQLite (better-sqlite3) to reliably store data offline. For the frontend design and analytics, we combined HTML, CSS, JavaScript, Chart.js, and jsPDF, making the application feel modern, fast, and visually stunning.

**What I Did:** I developed a complete, premium offline bill tracking and business analytics desktop application from scratch with an eye-catching design. I implemented complex local database features for managing bills and distributors efficiently, a secure local storage system that automatically manages financial records, and a comprehensive analytics dashboard with Chart.js to easily handle purchase tracking and generate automated PDF reports.
